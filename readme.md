# fields

A practical command line utility for filtering text streams.

## Features

-	Show certain lines only.
-	Show only words by their indices.
-	Use range expressions (1..20).

## Installation:

You will need a working go installation.

	go get -u -v github.com/insomnimus/fields/...

If above doesn't work, follow these steps:

-	`git clone https://github.com/insomnimus/fields`
-	`cd fields`
-	`go mod tidy`
-	`go install`

Insure your $GOBIN directory is under your $PATH.
